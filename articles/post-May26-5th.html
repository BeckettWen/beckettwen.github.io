MAY 5, 2026

iOS 26.5 RC Release

Today, Apple releases the iOS 26.5 RC (23F75), as the release candidate to the public release

As for my personal experience, iOS 26.5 is pretty good when talking about battery life or performance
This time, along side the improved battery performance, there are some new pride wallpapers added to the "Pride" collection
You can customize the color of the wallpaper aside from the 12 presets

For Functionalities, only the End to End Encryption (e2ee) for RCS message is now supported, though in beta phase

The public release is expected at the 12nd May, which may or maynot have the same build number
